## Who We Are

Newtype Industries builds **mission-critical AI and software systems for the modern battlefield**. Our team combines operational experience from the field with deep engineering expertise — so every line of code we ship is written with the end user in mind: the warfighter.

We don't build features for demos. We build capabilities that win fights.

## What We Build

- **Tactical C4I Systems** — Next-generation command, control, and communications tooling designed around real operator workflows, from Call-for-Fire to Radio Log.
- **On-Device AI** — Edge-optimized models (STT, vision, decision support) that run without a reliable network and without leaking data.
- **Defense-Grade Infrastructure** — Secure, auditable cloud and on-prem backends built to survive contested environments.

## How We Work

- **Operators in the loop.** Engineers work alongside people who have used these systems under pressure.
- **Ship narrow, ship real.** We'd rather deliver one capability that works downrange than ten that demo well.
- **Secure by default.** Security, auditability, and offline resilience are not afterthoughts.

## 회사 소개

뉴타입인더스트리즈는 **현대전을 위한 미션 크리티컬 AI · 소프트웨어 시스템**을 개발합니다. 현장에서의 작전 경험과 깊이 있는 엔지니어링 역량이 결합된 팀이, 최종 사용자인 전투원을 염두에 두고 모든 코드를 작성합니다.

우리는 데모를 위한 기능이 아닌, 전장에서 이기는 역량을 만듭니다.

## 우리가 만드는 것

- **전술 C4I 체계** — Call-for-Fire부터 무전 기록까지, 실제 운용자 워크플로우를 중심으로 설계된 차세대 지휘통제통신 도구.
- **온디바이스 AI** — 안정적인 네트워크 없이도, 데이터 유출 위험 없이 작동하는 엣지 최적화 모델 (STT · 비전 · 의사결정 지원).
- **국방급 인프라** — 경쟁 환경에서도 생존 가능한, 안전하고 감사 가능한 클라우드 · 온프레미스 백엔드.

## 일하는 방식

- **운용자와 함께.** 엔지니어는 실제 압박 속에서 이 시스템을 사용해 본 사람들과 함께 일합니다.
- **좁게, 실제로 출시.** 데모에서만 화려한 열 개보다, 현장에서 작동하는 하나를 출시하는 쪽을 택합니다.
- **기본값이 보안.** 보안, 감사 가능성, 오프라인 복원력은 뒤늦게 붙이는 것이 아닙니다.
